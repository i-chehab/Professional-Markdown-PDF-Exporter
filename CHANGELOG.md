# Changelog

All notable changes to **Professional Markdown PDF Exporter** are documented
in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/).

## 0.2.0

- Syntax highlighting for fenced code blocks (rendered at export time; no
  scripts run in the browser)
- Bundled Arabic fonts (Noto Naskh Arabic, Noto Kufi Arabic, SIL OFL) embedded
  into the document so Arabic output is identical on every machine
- Document title is now derived from the first `<h1>`, whether written as
  Markdown or as raw HTML (e.g. inside a cover page)

## 0.1.0

- Initial release
- Export Markdown to Arabic professional PDF
- Export Markdown to English professional PDF
- Context menu and Command Palette support
- System browser detection
- Local image support
- Professional document CSS presets
